# Money Transfer System

A production-grade digital money transfer microservice.

## Project Overview

**Project Type:** Enterprise Banking Microservice  
## Technology Stack

- **Version Control:** Git, GitHub
- **Backend:** Java 17, Spring Boot 3.x, MySQL 8.x
- **Frontend:** Angular 18+, TypeScript, Angular Material
- **Data Warehouse:** Snowflake Cloud
- **Testing:** JUnit 5

## Architecture

The system follows a microservices architecture with:
- RESTful API backend
- Single Page Application (SPA) frontend
- MySQL for transactional data
- Snowflake for analytics

## Project Structure

```
money-transfer-system/
├── backend/          # Spring Boot application
├── frontend/         # Angular application
├── database/         # MySQL scripts
├── snowflake/        # Snowflake scripts
├── docs/             # Documentation
└── README.md
```
